# Hanogt Codev

**HanogtCodev** - Kullanıcıların tarayıcıda veya masaüstünde diledikleri gibi kod yazıp çalıştırabilecekleri, açık kaynaklı, modern ve güçlü bir IDE.

🌐 **Canlı Site:** [hanogtcodev.com](https://hanogtcodev.com)

## 🚀 Özellikler

- **Çoklu Dil Desteği:** Python, JavaScript, C++, C#, Java, Go, PHP, Swift, Lua, HTML ve SQL
- **Gerçek Kod Çalıştırma:** Piston API ile güvenli sunucu ortamında kod çalıştırma
- **Modern Arayüz:** Dark/Light Mode
- **20 Dil Desteği:** Türkçe, İngilizce, Rusça, Azerbaycanca, İspanyolca, Kazakça, Japonca, Çince, Korece, Hintçe, Almanca, Nijeryaca, Fransızca, Belçikaca, Hollandaca, Lehçe, Norveççe, Fince, İsveççe, Yunanca
- **Masaüstü Uygulaması:** Windows, Linux ve macOS desteği (Electron)
- **AI Kodlama Asistanı:** Yapay zeka destekli kod yardımcısı
- **Hanogt Security Bot v2.0:** SQL Injection, XSS, Command Injection, Encoded Payload, Fileless Malware, Container Escape, DNS Exfiltration algılama, davranış analizi ve rate limiting
- **Mesajlaşma Sistemi:** Gerçek zamanlı mesajlaşma, sesli mesaj, sticker, mesaj düzenleme/silme, yanıtlama
- **Arkadaşlık Sistemi:** Arkadaş ekleme, engelleme, çevrimiçi durum, DND modu
- **Geri Bildirim Sistemi:** Kullanıcılar soru sorabilir, yorum yapabilir
- **%100 Ücretsiz:** Sınır yok, reklam yok

## 📦 Kurulum

### Gereksinimler
- [Node.js](https://nodejs.org/) v18+
- Git

### Hızlı Başlangıç
```bash
git clone https://github.com/Hanstudios1/HanogtCodev.git
cd HanogtCodev
npm install
npm run dev
```
Adres: `http://localhost:3000`

### Masaüstü Uygulaması
```bash
npm run dist
```
